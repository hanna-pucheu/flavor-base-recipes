# Flavor Base
A full-stack recipe discovery web application powered by Food.com data.
 
## Tech Stack
- **Frontend:** React
- **Backend:** Node.js
- **Database:** PostgreSQL
## Features
- Browse and search a large catalog of recipes sourced from Food.com
- Optimized query performance using CTEs, indexing, and materialized views — reducing runtimes from minutes to sub-second
## Dataset
This project uses the [Food.com](https://www.kaggle.com/datasets/shuyangli94/food-com-recipes-and-user-interactions) Recipes and Interactions dataset from Kaggle.
 
To run locally, download the following files and place them in the `data/raw/` directory:
- `RAW_recipes.csv`
- `RAW_interactions.csv`
